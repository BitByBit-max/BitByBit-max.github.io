Learn About Computers Bit By Bit

Bit by Bit is the fairy godmother for all new Epita students panicking over the seminary. Don’t worry, we’ll help you learn everything you need to know about OCaml in order to feel confident. With a game-like design, you will code many functions, fix bugs, and understand complexity as you unlock new levels. Make sure you can slay the boss at the end of the map !

About Us

Have the resources

There is so much information on python or C# but what about poor old OCaml ? Bit by Bit has got you covered. We give you everything you see in class and more.

Progress Faster

By getting more practice in coding and testing.Everything will start to fall into place in your head. You’ll be able to write any function your TD asks of you in no time.

Think for yourself

Using Bit by Bit will help you understand the thought process of writting your own function from A to Z. You finally won’t be paralyzed in front of an empty page.
